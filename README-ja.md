## name
stallionScrape

## Overview
This is scraping tool for [shuboba profile](http://keiba.no.coocan.jp/).  

## Requirement
Windows10 ~  

## Usage
1. Download zip or pull repository.
2. Execute below on cmd.
   ```
   npm install
   npm start
   ```
3. All finished, csv file will be in "csv" directory.
